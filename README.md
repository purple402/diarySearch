매년 내 맘에 쏙 드는 다이어리를 찾다가 결국 다이어리 검색을 위한 사이트까지 만들었습니다! 이렇게 만들었는데 내 맘에 드는 게 하나는 있겠지...?😂

사이트 사용법에 대한 자세한 설명은 아래 링크를 확인해주세요

-   기능

-   사용한 언어
    -   React
    -   HTML, CSS
